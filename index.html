Current Version: 1.0

Repository addon

Instructions for adding this repo:

  • Go to the Kodi file manager.
  • Click on "Add source"
  • The path for the source is https://weedoverfeed.github.io/repo/ (Give it the name "WeedOverfeed Github").
  • Go to "Addons"
  • In Addons, install an addon from zip. When it asks for the location, select "WeedOverfeed Github", and install repository.weedoverfeed-1.0.zip
  • Go back to Addons install, but this time, select "Install from repository"
  • Select the "WeedOverfeed's Repo"
  • Go into the "Program add-ons" section in the repo, and you'll find WeedOverfeed's Wizard

Enjoy!
